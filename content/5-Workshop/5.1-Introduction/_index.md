## 5.1. Introduction

This hands-on lab provides detailed guidelines on deploying the **Resume Fit** platform end-to-end on AWS. Resume Fit is an AI-powered web application evaluating compatibility score between Candidate Resumes (CVs) and Job Descriptions (JDs), aiming to facilitate efficient recruiting decisions.

The platform is designed using standard AWS reference architectures to guarantee high scalability and availability, including core components: virtual private network (VPC), elastic computing (EC2, Auto Scaling Groups), relational database (RDS PostgreSQL), object storage (S3), asynchronous message processing (SQS), system monitoring (CloudWatch), and security credentials management (Secrets Manager).
