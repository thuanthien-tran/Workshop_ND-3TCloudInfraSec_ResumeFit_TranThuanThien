### 5.7.1. Launch EC2 Backend Instance

1. Generate a new RSA **Key Pair** (`ResumeMatching-Key`) for secure SSH terminal logins.
   ![Create Key Pair](/images/5-Workshop/Tao_Key_Pair.jpg)

2. In the EC2 Console, launch a `t3.micro` instance named `ResumeMatching-Backend` in the private app subnet.
   ![Launch EC2 Instance](/images/5-Workshop/Tao_EC2_Backend.jpg)
