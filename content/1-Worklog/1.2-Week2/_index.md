### Week 2 Objectives:

* Master AWS IAM basics: Users, Groups, Policies, and Roles.
* Deploy and access Amazon EC2 instances via SSH.
* Configure Amazon S3 storage bucket and manage objects.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |
| 4 | - Study AWS Identity and Access Management (IAM). Practice creating IAM Users, User Groups, Policies (JSON structure), and IAM Roles. | 29/04/2026 | 29/04/2026 | [AWS IAM User Guide Introduction](https://docs.aws.amazon.com/IAM/latest/UserGuide/introduction.html) |
| 5 | - Learn Amazon EC2 concepts: instance types, AMI (Amazon Machine Image), Security Groups, and Key Pairs. | 30/04/2026 | 30/04/2026 | [AWS EC2 Concepts](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/concepts.html) |
| 6 | - Launch a Linux EC2 instance. Configure inbound Security Group rules. Connect to the instance using SSH from local machine. | 01/05/2026 | 01/05/2026 | [Connecting to Linux Instance Guide](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/AccessingInstances.html) |
| 2 | - Study Amazon S3 (Simple Storage Service) storage classes (Standard, IA, Glacier), bucket policies, and Lifecycle Rules. | 04/05/2026 | 04/05/2026 | [AWS S3 Intro Guide](https://docs.aws.amazon.com/AmazonS3/latest/userguide/Welcome.html) |
| 3 | - Hands-on S3: Create S3 bucket, upload objects, configure public access settings, and test access via S3 object URLs. | 05/05/2026 | 05/05/2026 | [Amazon S3 Getting Started](https://docs.aws.amazon.com/AmazonS3/latest/userguide/GetStartedWithS3.html) |


### Week 2 Achievements:

* Created non-root IAM admin users to follow security best practices.
* Successfully deployed an EC2 server and accessed it remotely via SSH.
* Learned S3 object storage management, permissions, and cost optimization via Lifecycle Policies.
