### Week 4 Objectives:

* Deploy and manage Amazon Elastic Block Store (EBS) volumes and Snapshots.
* Launch and connect to an Amazon Relational Database Service (RDS) instance.
* Monitor resources and set up automated alerts using Amazon CloudWatch.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |
| 4 | - Learn Amazon EBS architecture. Practice creating EBS volumes and attaching them to active EC2 instances. | 13/05/2026 | 13/05/2026 | [Amazon EBS Volume Types Guide](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/EBSVolumeTypes.html) |
| 5 | - Mount EBS volumes in Linux file systems. Learn how to take EBS snapshots and restore volumes from snapshots. | 14/05/2026 | 14/05/2026 | [EBS Volume Management Guide](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/AmazonEBS.html) |
| 6 | - Study Amazon RDS databases. Understand database engines, storage configurations, Multi-AZ deployments, and read replicas. | 15/05/2026 | 15/05/2026 | [AWS RDS Introduction Guide](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/Welcome.html) |
| 2 | - Deploy a MySQL/PostgreSQL database instance on RDS. Configure Security Groups to allow connection from the web server EC2 instance. | 18/05/2026 | 18/05/2026 | [Amazon RDS Getting Started Guide](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/CHAP_GettingStarted.html) |
| 3 | - Set up Amazon CloudWatch metrics monitoring (CPU, disk, memory). Create CloudWatch Alarms to trigger email alerts via Amazon SNS. | 19/05/2026 | 19/05/2026 | [Amazon CloudWatch Getting Started Guide](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/GettingStarted.html) |


### Week 4 Achievements:

* Successfully configured persistent external storage for virtual instances using EBS.
* Deployed a relational database using RDS and established secure connections with correct network firewalls.
* Configured real-time system monitoring and email notification alerts for automated resource management.
