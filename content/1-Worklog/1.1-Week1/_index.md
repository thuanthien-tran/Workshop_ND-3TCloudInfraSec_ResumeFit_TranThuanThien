### Week 1 Objectives:

* Understand the fundamentals of cloud computing (IaaS, PaaS, SaaS).
* Get familiar with AWS Global Infrastructure (Regions, AZs, Edge Locations).
* Set up and configure an AWS Free Tier account with baseline security.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |
| 2 | - Learn overview of cloud computing, deployment models (Public, Private, Hybrid), and service models (IaaS, PaaS, SaaS). | 20/04/2026 | 20/04/2026 | [What is Cloud Computing? - AWS](https://aws.amazon.com/what-is-cloud-computing/) |
| 3 | - Study AWS Global Infrastructure: Regions, Availability Zones (AZs), Local Zones, and Edge Locations. | 21/04/2026 | 21/04/2026 | [AWS Global Infrastructure Map](https://aws.amazon.com/about-aws/global-infrastructure/) |
| 4 | - Register and configure AWS Free Tier account. Set up MFA for Root account, create budget alerts, and explore Billing Dashboard. | 22/04/2026 | 22/04/2026 | [AWS Free Tier Details and Limits](https://aws.amazon.com/free/) |
| 5 | - Explore AWS Management Console interface. Learn how to search, navigate, and pin frequently used services. | 23/04/2026 | 23/04/2026 | [AWS Overview Whitepaper](https://docs.aws.amazon.com/whitepapers/latest/aws-overview/introduction.html) |
| 6 | - Study the Shared Responsibility Model and essential security concepts on AWS. | 24/04/2026 | 24/04/2026 | [Shared Responsibility Model](https://aws.amazon.com/compliance/shared-responsibility-model/) |


### Week 1 Achievements:

* Gained clear understanding of Cloud Computing concepts and service types (IaaS, PaaS, SaaS).
* Successfully created and secured AWS Free Tier account with Multi-Factor Authentication (MFA).
* Mastered navigating the AWS Management Console and understood the significance of Regions and AZs.
