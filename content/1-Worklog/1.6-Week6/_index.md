### Week 6 Objectives:

* Understand the concepts and benefits of Serverless architecture.
* Write, test, and deploy AWS Lambda functions.
* Expose Lambda functions via Amazon API Gateway.
* Trigger Lambda execution using S3 bucket events.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |
| 4 | - Study Serverless Computing architecture principles. Contrast serverless with server-based options. | 27/05/2026 | 27/05/2026 | [Serverless on AWS Details](https://aws.amazon.com/serverless/) |
| 5 | - Create first Lambda function using Python/Node.js on AWS Console. Learn environment variables and execution logs on CloudWatch Logs. | 28/05/2026 | 28/05/2026 | [AWS Lambda Dev Guide](https://docs.aws.amazon.com/lambda/latest/dg/welcome.html) |

### Week 6 Achievements:

* Understood pay-per-request execution billing models and compute savings on serverless workloads.
* Created event-driven automated pipelines connecting S3 object uploads with AWS Lambda processing.
* Designed and built secure REST endpoints with API Gateway forwarding inputs to background Lambda instances.
