# Harvest Report: "FCAJ Community"

### General Event Information
*   **Event Name:** FCAJ Community
*   **Date:** Academic Networking Event (Collaborated with HUTECH University and Swinburne University of Technology)
*   **Location:** Bitexco Financial Tower (AWS Vietnam Office)
*   **Organizers:** Team AWS G3 in collaboration with the First Cloud Journey Community
*   **Role:** Attendee

---

### Objectives & Goals
*   Share practical path guidance to becoming professional Cloud, Security, and DevOps Engineers.
*   Introduce emerging technology trends including Machine Learning (ML), Docker, and GraphRAG artificial intelligence.
*   Offer career orientation and discuss soft skills crucial for engineering teamwork efficiency.

---

### Speaker List
*   **Le Hoang Gia Dai:** Senior at HUTECH University, core member of Team AWS G3.
*   **Bao Huynh:** Junior Cloud Native Developer at Endava Vietnam and Founder/Head of ITea Lab.
*   **Tran Trung Vinh:** System Administrator at Central Retail Group.
*   **Nguyen Quoc Bao:** Technical Expert in Multiplayer Gaming and Cloud Computing.
*   **Truong Huy Phuoc:** Professional Teamwork Skills Trainer.
*   **Viet Phat:** AI Major Student at Swinburne University of Technology.

---

### Key Highlights

#### 1. Network Security powered by Machine Learning
*   **Topic:** *WAF + ML for Cyber Attack Detection - Machine Learning-based Network Intrusion Detection System (NIDS) on AWS*.
*   **Speaker:** Le Hoang Gia Dai.
*   **Details:** Discussed designing a Network Intrusion Detection System (NIDS) leveraging ML algorithms combined with AWS WAF to address limitations in traditional static rules. Used the academic CSE-CIC-IDS2018 dataset to train the model on real-world cyber threat patterns.
  
#### 2. Containerization Technology with Docker
*   **Topic:** *Docker – A containerization technology*.
*   **Speaker:** Bao Huynh.
*   **Details:** Shared details on Docker virtualization architecture, comparing VMs and Containers. Highlighted resources optimization, lightweight payloads, and the "Build once, run anywhere" philosophy.

#### 3. Career Roadmap from Helpdesk to Sysadmin
*   **Speaker:** Tran Trung Vinh.
*   **Details:** Covered the career path from IT Helpdesk to Senior Sysadmin. Stressed that mastering Linux administration and Networking fundamentals is mandatory before advancing to DevOps or Cloud Architecture.

#### 4. Multiplayer Game Development on Cloud
*   **Topic:** *Multiplayer in the Cloud - Connecting Godot Clients with AWS WebSockets*.
*   **Speaker:** Nguyen Quoc Bao.
*   **Details:** Explored utilizing AWS WebSockets alongside AWS Lambda and DynamoDB to enable low-latency, real-time communication between multiple Godot game clients.

#### 5. Next-generation Generative AI (GraphRAG)
*   **Topic:** *Build GraphRAG applications using Amazon Bedrock and Amazon Neptune*.
*   **Speaker:** Viet Phat.
*   **Details:** Explained GraphRAG (Graph Retrieval-Augmented Generation) architectures utilizing Amazon Bedrock foundation models and Amazon Neptune graph database to optimize LLM lookup accuracy.

#### 6. Efficient Teamwork Collaboration
*   **Speaker:** Truong Huy Phuoc.
*   **Details:** Shared 4 core rules for effective collaboration and introduced project management platforms like Trello and Slack to streamline developer pipelines.

---

### Key Takeaways

#### 1. Design Mindset
*   **Cloud Mindset:** Cloud computing is not just about renting servers, but designing auto-scaling, resilient architectures following the pay-as-you-go cost model.
*   **Data Integrity:** Input data quality determines the final accuracy and detection rate of Machine Learning security models.

#### 2. Technical Learnings
*   Understood how to handle imbalanced datasets during ML model training.
*   Acquired basic Docker CLI commands and serverless WebSockets connection flows on AWS.

#### 3. Operational Best Practices
*   **Deployment Rule:** Never test configuration parameters directly on a live Production environment to preserve system uptime.
*   Automate repetitive workflows and document all configuration pipelines.

---

### Applying to Work
*   **Project Implementations:** Containerize the **AI Resume Matching** application code using Docker to ensure local-to-production consistency. Bind AWS WAF to the ALB endpoints to safeguard against common web threats like SQL Injection and Cross-Site Scripting (XSS).
*   **Next Technologies to Experiment with:** Test Amazon Bedrock API integrations to enhance the CV scanning and interview question generation accuracy of Resume Fit.
*   **Workflow Enhancements:** Build real-time CloudWatch dashboards to increase threat visibility across networks.

---

### Event Experience
*   **Interactive Demos:** Witnessed live demonstrations of NIDS threat detection, Docker containers in action, and real-time connectivity between Godot clients.
*   **Professional Networking:** Interacted and networked via LinkedIn with senior speakers working at multinational enterprises (Endava, Central Retail).
*   **Most Impressive:** The synergy between AWS services and ML/AI frameworks to solve complex, real-world security and operations problems.

---

### Lessons Learned
*   **ML for Threat Mitigation:** Traditional signature-based security rules are inadequate against Zero-day threats; integrating ML is crucial to detect anomalous behaviors.
*   **Hands-on Project Priority:** Your academic starting line is secondary to consistent hands-on lab practice and building physical portfolios rather than focusing solely on theoretical certifications.
*   **Career Focus:** Master 1-2 core technical domains before expanding, and showcase your expertise through successfully completed projects.
